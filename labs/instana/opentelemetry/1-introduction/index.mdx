# Introduction

In this lab of the Instana Jam-in-a-box series you will be going through
several key exercises that will help you learn more about application
monitoring using OpenTelemetry. You will cover topics such as:

- Installing an Instana agent in the OpenShift cluster.
- Working with an OpenTelemetry demo application configured to support Instana monitoring.
- Creating a 'Website' in Instana for End User Monitoring.
- Exploring OpenTelemetry metrics and traces in Instana
