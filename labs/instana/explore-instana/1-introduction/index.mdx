# Introduction

In the Instana portion of the Tech Jam lab, you will be going through several
key exercises in order to learn more about Instana's capabilities. The lab
should be executed in the numbered order that you see on the left side of the
screen. Some of the sections have dependencies on the previous section.

- First, you will be learning about Instana's Kubernetes and Infrastructure
  Monitoring capabilities
- Then, you will learn how to create Application Perspectives for demo apps that
  have been pre-installed so you scope your views of the applications.
- Next, you will explore the Application Perspectives that you created.
- Next, you will learn more about Website and Mobile App monitoring.
- Finally, you will briefly cover Instana's Unbounded Analytics.
