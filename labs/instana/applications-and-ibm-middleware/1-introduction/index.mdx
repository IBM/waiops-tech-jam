# Introduction

In the second lab of the Instana Jam-in-a-box series you will be going through
several key exercises that will help you learn more about application
monitoring. You will cover topics such as:

- Creating a 'Website' in Instana for End User Monitoring.
- Working with a Demo application configured to support Instana Website
  monitoring.
- Introducing load to the demo application and observing the results in Instana.
- Configuring IBM MQ and ACE for Instana monitoring and tracing.
