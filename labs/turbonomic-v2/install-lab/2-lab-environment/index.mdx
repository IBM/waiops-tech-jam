# Lab Environment

In this lab you will have access to two Openshift clusters:

1. **AIOps cluster** - A full installation of OCP that will be used to install
   Turbonomic. Instana and the Instana Agent has already been pre-installed
   ready for integration with Turbonomic.
2. **Apps cluster** - An instance of Single Node OpenShift that has an Instana
   Agent and the demo applications _RobotShop_ and _Quote Of The Day_
   pre-installed and sending metrics to Instana.

In this lab you will install the Turbonomic Server on the AIOps OCP Cluster and
integrate Turbonomic with the Instana Server also running on the AIOps Cluster.

![architecture](images/architecture-instana-turbo.png)

## Prerequisites

To complete this lab you will need:

- Instana License
  - Agent Key - Entered when requesting lab environment
  - Sales Key - Entered when requesting lab environment
- Turbonomic License key - Required during lab
